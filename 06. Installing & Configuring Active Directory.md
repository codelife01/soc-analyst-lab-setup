# Install and Configure Active Directory

## Install Active Directory

Change the name of the server by typing in name in the search window.

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image1.webp)

Select Rename this PC and type in the name of the server. 

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image2.webp)

Restart the server.

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image3.webp)

After the server restarts select Manage and then “Add Roles and Features”.

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image4.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image5.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image6.webp)

Select Next

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image7.webp)

Select “Active Directory Domain Services”.

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image8.webp)

Select “Add Features”

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image9.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image10.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image11.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image12.webp)

Check the Restart if needed box and select Yes

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image13.webp)

Select Install

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image14.webp)

It may take a few minutes

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image15.webp)

When it finished installing, select promote this server to a domain controller

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image16.webp)

Check the “Add a new forest box, type in the Root domain name, then select Next.

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image17.webp)

Type in passwords. It can be the same password you originally set up for the VM. Then select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image18.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image19.webp)

After a few seconds the NetBIOS domain name will auto-populate and then select Next.

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image20.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image21.webp)

Select Next

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image22.webp)

After the prerequisites check is complete, select Install.

![Untitled](Screenshots/06-Installing&ConfiguringActiveDirectory/image23.webp)

After install completes, you will be presented with the screenshot below before it restarts. Select Close.

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image24.webp)

## Setting Network Connections

Right click the network icon in the sytem tray and select Open Network & Internet settings.

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image50.webp)

Choose Ethernet.

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image51.webp)

Select Edit IP Assignement

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image52.webp)

Make the following selections.

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image53.webp)

Bring up the command prompt again and type in ipconfig /all. 

![image.png](Screenshots/06-Installing&ConfiguringActiveDirectory/image54.webp)

As you can see above, the IP address, Gateway, and DNS is correct.

Congrats! You have completed the initial Active Directory and Certificate Services configuration.
